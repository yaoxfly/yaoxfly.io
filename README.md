## Introduction
personal website based on vitepress

## Use 

Run

```js
pnpm run dev
```

Pack

```js
pnpm run  build
```

Preview

```js
pnpm run serve 
or
pnpm run preview // Run pnpm run build and then run pnpm run serve
```

> The preview operation can only be performed after packaging
