## 简介
基于vitepress的个人网站，记录自己的奇思妙想。

## 使用

运行

```js
pnpm run dev
```

打包

```js
pnpm run  build
```

预览

```js
pnpm run serve 
or
pnpm run preview // 当前命令会先执行pnpm run build 再执行 pnpm run serve 
```

> 打包后才能执行预览操作
